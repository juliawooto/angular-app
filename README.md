# Angular example application with REST API web service
This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 6.0.8.


## Technologies and packages
* node 8.11.3
* npm 6.5.0
* Angular 6
* Bootstrap 4.1.1
* font-awesome 4.7.0
* json-server 0.12.1
* jsonwebtoken 8.1.1
* Typescript 2.7.2   
* rxjs 6.0.0


## Run application
Command prompt 1:
CMD: npm run json

Command prompt 2:
CMD: ng serve --port 3000 --open
